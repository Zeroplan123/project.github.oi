    JavaScript adalah bahasa pemrograman tingkat tinggi dan terinterpretasi yang dikenal terutama karena perannya dalam pengembangan web. Ini adalah salah satu teknologi inti untuk membangun situs web interaktif dan dinamis. JavaScript memungkinkan pengembang membuat skrip sisi klien yang berjalan dalam browser web untuk memanipulasi Document Object Model (DOM), menangani interaksi pengguna, dan memperbarui konten halaman web secara real-time tanpa perlu me-refresh halaman.
